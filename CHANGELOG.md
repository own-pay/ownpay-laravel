# Changelog

All notable changes to `ownpay-laravel` will be documented in this file.

## 1.0.0 - 2026-07-16

### Added

- Initial release
- Payment creation and retrieval
- Transaction listing and filtering
- Refund processing
- Customer management
- Webhook signature verification
- HMAC-SHA256 webhook signing
- Bearer token authentication
- Rate limit handling
- Idempotency key support
- PHP 8.3+ enums and value objects
- Money value object with arithmetic
- Payment status, transaction status, and refund status enums
- Webhook event types
- Laravel service provider with auto-discovery
- Facade for convenient access
- Webhook middleware for signature verification
- Artisan commands for testing connection and verifying webhooks
- Database migration for webhook logs
- Comprehensive test suite
- PHPStan level 8 static analysis
- Laravel Pint code formatting
- PHPUnit configuration
